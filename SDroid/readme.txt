                      SDroid Library
------------------------------------------------------------
SDroid is a  framework  released under the MIT  license, for 
designing and running custom bots for Steam network  capable
of trading and managing the Steam Account they are connected
to.

Based on geel9/SteamAuth and Jessecar96/SteamBot projects

For more  information about this  library,  please check out
the project's GitHub page:
https://github.com/falahati/SDroid
