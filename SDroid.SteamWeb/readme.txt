                 SDroid.SteamWeb Library
------------------------------------------------------------
SDroid.SteamWeb is a library released under the MIT license, 
that  provides  Steam  web  login,  web  access  and  WebAPI
functionalities.

For more  information about this  library,  please check the
project GitHub page:

https://github.com/falahati/SDroid/tree/master/SDroid.SteamWeb
